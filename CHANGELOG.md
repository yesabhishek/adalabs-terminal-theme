# Changelog

## 1.1.0

- Replaced the six branded dark variants with ten simple shade-based themes: **AdaLabs Dark 1-5** and **AdaLabs Light 1-5**.
- Added a consistent syntax palette across every shade, with improved colors for filenames, line numbers, snippets, Markdown/code blocks, diagnostics, git states, and semantic tokens.
- Updated the dark range to use neutral black and graphite workbench accents instead of blue-tinted chrome.
- Updated README and extension metadata to describe the dark and light ranges for OLED and LCD/TN displays.

## 1.0.0

- Renamed the extension and theme family to **AdaLabs Terminal**.
- Expanded all six themes with complete workbench, terminal, diagnostics, git, diff, peek, minimap, quick pick, notification, and editor widget colors.
- Added broader TextMate token coverage and semantic token colors.
- Added a Marketplace-ready icon, README, license, and packaging ignore rules.
